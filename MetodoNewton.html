Implementación del Método de Newton-Raphson

Este método es utilizado para encontrar raíces de funciones diferenciables. el método de Newton-Raphson implica emplear la tangente a la curva de la función en un punto específico con el fin de obtener una aproximación mejorada de la raíz. La fórmula general que rige este método se expresa de la siguiente manera :

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

\(x_{n+1}\) es la nueva aproximación de la raíz.

\(x_n\) es la valor aproximado actual de la raíz

\(f(x_n)\) es el val de la función en \(x_n\)

\(f'(x_n)\) es la derivada de la función en \(x_n\)

Código del programa en Python


import math as m
import matplotlib.pyplot as plt
import numpy as np
#Metodo de las tangentes de Newton
#se utiliza para aproximar a una raiz por medio de una recta tangente

# Requerimientos
#Funcion (continua)
#Derivada de la funcion
# Metodo de Newton   Xn+1 = Xn - ( f(Xn)/ f'(Xn) )

#funciones
def fx(x):
    #Funcion
   return x**2-2
  #  return x*m.sin(x)+x**3
def dfx(x):
    #derivada de la funcion
    return 2*x
    #return (x*m.cos(x)+ m.sin(x))  + 3*x**2


# Declare un ciclo for con 5 iteraciones
#Entre mayor sea el número se aproxima más a la raiz aunque se acerca a un limite pronto
X0 = 1
for i in range(5):
    X0 = X0 - ( fx(X0)) / (dfx(X0))  
    print(X0)

#Codigo de la grafica
X = np.arange(-2,2,0.01)
Y = np.array([fx(i) for i in X ])

plt.plot(X,Y)
plt.grid()
plt.axhline(y=0, color="black", linestyle="--")
plt.axvline(color="grey")
plt.text(1, 8, "x$^2$ +2")
plt.plot(X0,0,"o")
plt.title("Metodo tangentes de Newton")

plt.annotate("Raiz ({0:.4f}".format(X0)+f",0)", (X0,0),textcoords="offset points",xytext=(-5,10),ha='center')
plt.show()
